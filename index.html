Overview

PapAiEra is a specialized Python library engineered for the Pulp and Paper industry. It mathematically models and solves optimization problems based on the Best Available Techniques (BAT) reference documents (EU BREF 2015).

✓
Comprehensive coverage of pulping, papermaking, and sustainability workflows
✓
AI-powered optimization using SciPy constraints and Differential Evolution
✓
Physics-aware digital twins for boiler and dryer hood optimization
✓
ML-based prediction models with industrial heuristics
✓
ABB-compliant Variance Partition Analysis (VPA) engine

Installation

Install PapAiEra via pip in your Python environment:

pip install PapAiEra

Requirements: Python 3.8+, NumPy, SciPy, scikit-learn

✓
No system-level dependencies required
✓
Pure Python implementation with optimized C extensions

Advanced Usage Examples

🧠 Universal Bleaching Optimizer

Define custom bleaching stages with dynamic chemical optimization using Differential Evolution.

✓
Dynamic Sequences: Support any stage configuration (D0-EOP-D1, O-Z-P, etc.)
✓
AI Simulation: Plug-in system for Heuristic, PNN, or Regression models
✓
Global Optimization: Minimize chemical cost while hitting quality targets
from pap_ai_era.bleaching import run_optimization_from_config config = { "stages": [ { "name": "D0", "chemicals": [{"name": "ClO2", "min_dosage": 5, "max_dosage": 30, "cost": 0.8}] }, { "name": "EOP", "chemicals": [ {"name": "NaOH", "min_dosage": 5, "max_dosage": 25, "cost": 0.4}, {"name": "H2O2", "min_dosage": 2, "max_dosage": 15, "cost": 0.6} ] } ], "targets": {"brightness": 90.0, "kappa": 2.0, "viscosity_min": 800} } result = run_optimization_from_config(config)

📊 Variance Partition Analysis (VPA)

ABB-compliant 2-Sigma analysis decomposing reel variability into Machine Direction (MDL), Cross Direction (CD), and Residual (MDS) components.

import numpy as np from pap_ai_era.papermaking.variability_analysis import compute_vpa # 50 scans × 600 data boxes data = np.random.normal(loc=50, scale=1.5, size=(50, 600)) vpa_report = compute_vpa(data, process_average=50.0) print(f"Total 2-Sigma Variability: {vpa_report.normalised['TOT']:.2f}%") print(f"Primary Problem Detected: {vpa_report.primary_problem}")

📚 ML Bulk Prediction Model

Physics-aware Random Forest regression for predicting paper/board bulk (cm³/g) with layer-wise furnish analysis.

✓
Multi-Layer Support: Independent pulp blends for Top/Middle/Bottom layers
✓
Hybrid Modeling: Combines ML with industrial heuristics for robustness
✓
Process Sensitivity: Track impact of nip load, refining (CSF) on sheet thickness
from pap_ai_era.papermaking import BulkModel # Define layer structure and process conditions layers = [ {"gsm": 50, "sw_ratio": 0.9, "csf": 550}, # Top layer {"gsm": 150, "sw_ratio": 0.1, "csf": 400} # Middle layer ] process = {"nip_load": 60, "n_nips": 2} model = BulkModel(n_layers=2) # model.train(historical_data) # Optional: train on mill data predicted_bulk = model.predict(layers, process)

⚙️ Kraft Digester H-Factor Calculator

Calculate integrated cooking reaction rate using the Vroom equation to hit target Kappa numbers.

from pap_ai_era.pulping.kraft import calculate_h_factor # 165°C cooking zone for 45 minutes h_factor = calculate_h_factor(temperature_celsius=165.0, time_minutes=45.0) print(f"H-Factor Generated: {h_factor:.2f}")

API Reference

calculate_h_factor(temperature_celsius: float, time_minutes: float) → float

Computes the Vroom H-factor for kraft pulping based on temperature and time.

Parameters: temperature_celsius (float): Cooking temperature | time_minutes (float): Residence time
Returns: float - Integrated H-factor value

run_optimization_from_config(config: dict) → OptimizationResult

Executes global optimization for bleaching sequences using Differential Evolution.

Parameters: config (dict): Dictionary with stages, chemicals, and targets
Returns: OptimizationResult with optimal dosages, cost, and convergence status

compute_vpa(data: np.ndarray, process_average: float) → VPAReport

Performs Variance Partition Analysis decomposing variability into MDL, CD, and MDS components.

Parameters: data (ndarray): 2D array of shape (scans, data_boxes) | process_average (float): Target process mean
Returns: VPAReport with normalized variances and root-cause inference

BulkModel.predict(layers: list, process: dict) → float

Predicts paper bulk using physics-aware ML model with layer-wise furnish inputs.

Parameters: layers (list): List of dicts with gsm, sw_ratio, csf | process (dict): Dict with nip_load, n_nips
Returns: float - Predicted bulk in cm³/g

run_hood_optimization(params: dict) → dict

Optimizes paper machine dryer hood balancing evaporation load against energy consumption.

Parameters: params (dict): Machine speed, GSM, width, moisture targets, ambient conditions
Returns: dict with optimal fan RPM, recirculation ratio, and energy savings

Troubleshooting

⚠️ Optimization returns success: False

Target constraints may be mathematically impossible given the current bounds.

Solution: Adjust boundary values in the bounds parameter or relax quality targets.

❌ Import Errors

Module not found after installation.

Solution: Ensure you run pip install PapAiEra in your active virtual environment. Verify with pip list | grep PapAiEra.

📐 VPA Data Shape Error

Variance Partition Analysis expects specific input dimensions.

Solution: Provide a 2D NumPy array of shape (scans, data_boxes). Example: (50, 600) for 50 machine scans with 600 measurement points each.
