ML Model Bulk Prediction with Multi-Layer Support

from pap_ai_era.papermaking import BulkModel # Define multi-layer sheet structure layers = [ {"gsm": 50, "sw_ratio": 0.9, "csf": 550}, # Top layer {"gsm": 150, "sw_ratio": 0.1, "csf": 400} # Bottom layer ] process = {"nip_load": 60, "n_nips": 2} # Initialize and predict (model can be trained on historical mill data) model = BulkModel(n_layers=2) # model.train(historical_data) predicted_bulk = model.predict(layers, process) print(f"Predicted Bulk: {predicted_bulk:.3f} cm³/g")

Analytics Variance Partition Analysis (2-Sigma)

import numpy as np from pap_ai_era.papermaking.variability_analysis import compute_vpa # Simulated scan data: 50 scans × 600 cross-direction boxes data = np.random.normal(loc=50, scale=1.5, size=(50, 600)) # Compute VPA report with process average reference vpa_report = compute_vpa(data, process_average=50.0) print(f"Total 2-Sigma Variability: {vpa_report.normalised['TOT']:.2f}%") print(f"Primary Problem Detected: {vpa_report.primary_problem}") # Output: Total 2-Sigma Variability: 6.12% # Primary Problem Detected: Cross Direction Profile

🔧 Core Imports

from pap_ai_era.pulping import kraft from pap_ai_era.papermaking import machine, BulkModel from pap_ai_era.sustainability import emissions from pap_ai_era.optimization_models import *

🎯 Key Parameters

  • temperature_celsius - Process temperature
  • time_minutes - Reaction duration
  • target_gsm - Paper basis weight
  • moisture_pct - Water content target
  • chemical_dosage - Additive amounts

📦 Output Types

  • dict - Detailed results with metrics
  • float - Single calculated values
    • dataclass - Structured optimization reports
  • numpy.array - Variability analysis matrices
